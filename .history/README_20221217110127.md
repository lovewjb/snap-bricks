**用法**：将项目克隆至本地:

```
git clone https://github.com/lovewjb/snap-bricks.git
```

或者

```
git clone git@github.com:lovewjb/snap-bricks.git (需配置)
```

然后使用 npm 包管理器（需安装 Node.js, 没安装的去[官网](https://nodejs.org)下载安装包）执行以下命令：`npm install`即可编译。
